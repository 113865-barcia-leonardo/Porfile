Sobre mí

Hola, soy Leonardo, tengo 28 años y soy un apasionado del mundo de la programación. Actualmente estudio la Tecnicatura Universitaria en Programación en la Universidad Tecnológica Nacional (FRC), donde ya completé el primer año de formación.

En mi trayectoria he profundizado en Java y Spring Boot, trabajando con herramientas como Maven, JUnit y Mockito para realizar pruebas unitarias robustas. Recientemente amplié mis conocimientos hacia el frontend con Angular, donde aprendí a trabajar con TypeScript, componentes, routing para navegación entre páginas, y el manejo de Observables y Subscripciones para gestión de eventos reactivos.

También domino Tailwind CSS para crear interfaces modernas y responsivas, complementando mis habilidades en HTML5, CSS y JavaScript. Además, utilizo GitHub como parte esencial de mi flujo de trabajo y tengo experiencia con bases de datos como SQL Server.

Disfruto del trabajo en equipo y creo firmemente que la colaboración y la comunicación son claves para lograr buenos resultados. Me caracterizo por ser proactivo, con muchas ganas de aprender, enfrentar nuevos desafíos y seguir mejorando día a día.
